Shell Configuration
-------------------

1.pip install django_extensions
add it in the   INSTALLED_APPS

2.Set the default shell:

You can choose between different shells like IPython or bpython. Hereâ€™s how to set it:
SHELL_PLUS = "ipython"  # Options: "ipython", "bpython", or "python"

3.Auto-import models:

By default, shell_plus auto-imports all models. If you want to customize this behavior, you can use:
SHELL_PLUS_AUTOIMPORT = True  # Set to False to disable auto-importing

4.Pre-import specific modules:

If you want to auto-import specific modules or functions when starting the shell, you can do so with:
SHELL_PLUS_PRE_IMPORTS = [
    ('myapp.models', 'MyModel'),  # Import MyModel from myapp.models
    ('myapp.utils', 'my_function'),  # Import my_function from myapp.utils
]

5.Enable SQL printing:

If you want to see the SQL queries generated by your ORM, you can enable it with:
SHELL_PLUS_PRINT_SQL = True  # Set to False to disable

Additional Options
6.Custom startup script:

You can specify a custom startup script that runs every time you start shell_plus:
SHELL_PLUS_POST_IMPORTS = [
    'from myapp.utils import my_function',
]

7.Custom shell configuration:

If you want to pass custom arguments to your shell, you can do so with:
SHELL_PLUS_ARGUMENTS = ['--simple-prompt']  # Example argument for IPython
